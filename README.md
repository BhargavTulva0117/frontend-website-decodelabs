# frontend-website-decodelabs
🐾 Aureus Nutricare – Veterinary Products Website
📌 Project Overview

Aureus Nutricare is a responsive veterinary healthcare and animal nutrition website developed as part of the Full Stack Development Internship Project 1.

The project focuses on creating a modern, responsive, and user-friendly frontend interface using HTML, CSS, and JavaScript. The website showcases veterinary products, company information, services, product categories, testimonials, and contact details while providing an engaging user experience across desktop, tablet, and mobile devices.

🎯 Project Objective

The primary objective of this project is to:

Design a fully responsive frontend interface.
Implement modern UI/UX principles.
Create multiple interconnected webpages.
Provide smooth navigation and user interaction.
Demonstrate frontend development skills using HTML, CSS, and JavaScript.
