# 🐄 Aureus Nutricare – Veterinary Products Website
A modern, responsive veterinary healthcare and animal nutrition website built using **HTML, CSS, and JavaScript**. The project focuses on delivering a professional digital experience for veterinary product manufacturers, distributors, and farmers.
## 🚀 Key Features
- Responsive design for Desktop, Tablet, and Mobile devices
- Modern UI/UX with clean and professional layouts
- Product Catalog with dedicated Product Details pages
- Product Search and Category Filtering
- Interactive Product Slider
- Mobile Navigation Menu
- Animated Sections using Scroll Effects
- Loading Screen Animation
- Contact Section and Business Information
- WhatsApp Quick Connect Integration
- Optimized and Reusable Component-Based Design
## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts
